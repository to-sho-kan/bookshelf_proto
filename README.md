Installation
============
Install rails.

```
$ bundle install
```


Install gulp, bower, etc.

```
$ cd ngapp/
$ npm install -g gulp bower
$ npm install
$ bower install
```

Run application.

```
# in Rails root
$ rails s

# in ngapp/
$ cd ngapp/
$ gulp build
$ gulp
```

Deploy
======
/publicにフロントエンド側のファイルを作成する
```
$ cd ngapp/
$ gulp prod
```

/publicのフロントエンド側のファイルを削除する
```
$ cd ngapp/
$ gulp clean
```
